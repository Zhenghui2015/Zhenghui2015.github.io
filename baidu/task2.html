Requirement

In this task, user can input city's AQI. After validation, the result will show on a list. Detailed requirements are listed below:

  1. City name and AQI should removed extra space
  2. There will be alert for unvalidate input
  3. User can click "delete" button to remove a row

note

  1. use addEventListener() to bind function to a event
  2. use dataset to define data needed in DOM
  3. dynamic generate table and delete rows
